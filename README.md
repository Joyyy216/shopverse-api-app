# 🛒 ShopVerse REST API App

## 📖 Deskripsi Aplikasi

ShopVerse adalah aplikasi React Native (Expo) yang menggunakan **FakeStore API** untuk menampilkan daftar produk. Aplikasi ini memiliki fitur pencarian, pull-to-refresh, detail produk, serta menampilkan kondisi loading, success, dan error.

---

## 🌐 API yang Digunakan

FakeStore API

https://fakestoreapi.com/products

---

## ✨ Fitur

### Level 1

* ✅ Fetch REST API menggunakan Axios
* ✅ Async/Await
* ✅ useEffect
* ✅ Try/Catch/Finally
* ✅ Loading State
* ✅ Success State
* ✅ Error State + Retry
* ✅ FlatList

### Level 2 (Dipilih)

* ✅ Search Product
* ✅ Pull-to-Refresh
* ✅ Detail Product (Modal)
* ✅ Empty State

## 🚀 Cara Menjalankan

1. Clone repository

```bash
git clone https://github.com/Joyyy216/shopverse-api-app.git
```

2. Masuk ke folder project

```bash
cd shopverse-api-app
```

3. Install dependency

```bash
npm install
```

4. Jalankan aplikasi

```bash
npx expo start
```

5. Scan QR Code menggunakan Expo Go.
